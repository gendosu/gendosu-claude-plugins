# Quick Reference

## General Guidelines

- **README files**: Maintain both English (README.md) and Japanese (README.ja.md) versions
- **Plugin changes**: Update version in plugin.json and document in CHANGELOG.md
- **Versioning**: Follow semantic versioning for all plugins
